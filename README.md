# LME Copper Forward Curve Analytics
Ứng dụng phân tích chuyên sâu đường cong giá đồng LME bằng Python & Streamlit.

## Tính năng chính
* Trực quan hóa đường cong giá (Forward Curve) qua thời gian.
* Phân tích thống kê, Histogram & Volatility.
* Phân tích tương quan (Correlation Matrix) & PCA (Level, Slope, Curvature).
* Xuất dữ liệu ra Excel chuẩn hóa.

## Cách chạy ứng dụng
1. Cài đặt thư viện: `pip install -r requirements.txt`
2. Chạy app: `streamlit run app.py`
